Square - создает квадрат, строит на нем сетку из треугольников, 
выводит в файл (out_2D) информацию о каждом треугольнике сетки:
	- номер треугольника и номера его узлов
	- координаты узлов
	- координаты центра
	- список номеров соседних треугольников
Нумерация треугольников начинается не с 1 потому что в gmsh сквозная нумерация всех элементов.

Cube - всё то же самое, только для кубика и трёхмерной сетки из тетраэдров

Arc - строит сетку на импортированной модели (простая арка), всё точно так же как в Cube
